# Flwing (Fly + Wing)

**Flwing**은 Rust로 작성된 고성능 네이티브 벡터 그래픽 디자인 애플리케이션입니다. Adobe Illustrator의 핵심 기능을 Rust의 안전성과 GPU 가속의 성능으로 재해석했습니다.

## 🚀 주요 기능

- **강력한 렌더링 엔진**: `wgpu`와 `lyon`을 사용한 GPU 가속 벡터 렌더링.
- **고급 그라디언트**: 선형, 방사형 그라디언트는 물론 전문적인 **자유 메쉬(Mesh) 그라디언트** 지원.
- **전문 타이포그래피**: `ttf-parser`와 `swash` 기반의 정밀한 텍스트 셰이핑, 자간/커닝 제어 및 **텍스트 윤곽선 변환** 기능.
- **이미지 트레이스**: 래스터 이미지를 벡터 경로로 자동 변환하는 `vtracer` 통합.
- **독자적 바이너리 포맷**: 폰트 임베딩을 지원하는 고성능 `.flw` 포맷.
- **다국어 지원**: 시스템 언어에 따른 한국어/영어 UI 자동 전환.

## 🛠 실행 방법

애플리케이션을 실행하려면 Rust 툴체인이 설치되어 있어야 합니다.

```bash
# 프로젝트 디렉토리로 이동
cd Flwing

# 애플리케이션 실행
cargo run --release
```

디버그 모드에서 실행하려면 다음과 같이 입력하세요:
```bash
cargo run
```

## 📂 프로젝트 구조

- `src/core`: 벡터 데이터 모델 및 수학적 기본 타입
- `src/renderer`: GPU 가속 렌더링 파이프라인
- `src/text`: 타이포그래피 엔진 및 폰트 관리
- `src/trace`: 이미지 벡터화 엔진 연동
- `src/format`: `.flw` 바이너리 직렬화/역직렬화
- `src/ui`: egui 기반 사용자 인터페이스
- `src/i18n`: 다국어 지원 모듈
